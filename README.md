A Claude skill and standalone web app that clears brain fog and resets focus through cognitive mini-games ‚Äî back to productive flow in 10 minutes.

## The idea

When brain fog hits mid-deep-work, productivity tips don't help. What actually works is shifting your brain from open-ended, high-load thinking into narrow, closed-loop pattern recognition ‚Äî the same mental state you get from assembling LEGO or solving a puzzle.

brain-reset-skill is a cognitive fidget toy. It gives your overloaded brain a real break.

---

## What's included

### Claude skill (`brain-reset.skill`)
Install into Claude and trigger a structured 10-minute reset session from any conversation. The session runs 3 rounds of increasing difficulty ‚Äî easy, medium, hard ‚Äî with 5 challenges served as batches so you stay in flow.

### Brain reset hub (`brain-reset-hub.html`)
A standalone web app with 6 playable games you can open directly in any browser ‚Äî no install, no login.

---

## The 6 games

| Game | Cognitive mode |
|------|---------------|
| Sudoku | logical deduction |
| Nonogram | spatial pattern recognition |
| Minesweeper | probabilistic reasoning |
| Sliding Puzzle | spatial planning |
| Math Chain | working memory + arithmetic |
| Countdown | creative problem solving with numbers |

Each game has multiple difficulty levels and a how-to built in.

---

## 10-minute session structure (Claude skill)

**Round 1 ‚Äî warm up (easy, ~3 mins)**
2 challenges served as a batch. Different types. Low stakes.

**Round 2 ‚Äî build (medium, ~3 mins)**
2 challenges. Slightly more cognitive load.

**Round 3 ‚Äî focus (hard, ~3 mins)**
1 hard challenge. Full attention required.

Closes with a single line: *you're back.*

## How to use

### Option A ‚Äî Claude skill
1. Download `brain-reset.skill`
2. In Claude, go to **Settings ‚Üí Skills**
3. Upload the file
4. Say `brain reset`, `I can't focus`, or `need a mental break`

### Option B ‚Äî standalone web app
1. Download `brain-reset-hub.html`
2. Open it in any browser
3. Pick a game and play

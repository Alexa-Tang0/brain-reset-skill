# 🧠 brain-reset-skill

A Claude skill that delivers structured 2–5 minute cognitive micro-challenges to clear brain fog and reset focus during work breaks.

## What it does

Instead of giving advice or tips, this skill drops you straight into a finite, low-stakes puzzle. Think of it as a cognitive fidget toy.

## How to trigger it

Say any of these to Claude:
- brain reset
- I can't focus
- need a mental break
- clear my head
- I'm foggy

## Challenge types

Odd One Out, Pattern Completion, Spot the Difference, Category Sort, Reconstruct the Rule, True/False Cluster, Mental Math Chain.

Difficulty defaults to medium. Ask for harder or easier after each round.

## Installation

1. Download brain-reset-skill.skill
2. In Claude, go to Settings → Skills
3. Upload the .skill file
